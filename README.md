# Connect Four

Simple two-player *Connect Four* CLI game for [The Odin Project](https://www.theodinproject.com/lessons/ruby-connect-four#project-tdd-connect-four)

## Installation

```{bash}
$ git clone https://github.com/jalalk97/connect-four
```

## Usage

```{bash}
$ cd connect-four
$ bundle install
$ bundle exec ruby main.rb
```
